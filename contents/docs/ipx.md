To Install using [crates.io](https://crates.io)

> [!install] cargo install ipx

Basic usage:

```shell
ipx example.com
```

Dependencies:

```toml
ipgeolocate = "0.3.6"
tokio = { version = "1.37.0", features = ["full"] }
```
